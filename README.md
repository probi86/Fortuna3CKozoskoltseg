# Fortuna 3C — Közösköltség összesítés

A **3C tömb** havi közös költségeinek áttekintő táblázata és grafikonja.

## ➡️ Az oldal: https://probi86.github.io/Fortuna3CKozoskoltseg/

- Csak a **közös** tételek szerepelnek — a víz, melegvíz, csatorna és a fűtés-fogyasztás (egyéni, saját mérőóra szerinti) **nincs** benne.
- Csak **épület-szintű** összegek, lakásonkénti bontás és személyes adat nélkül.
- A keresőkben nem jelenik meg; a linket a lakókkal érdemes megosztani.
